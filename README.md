# rockychat

### Requirements:
Node v20+
npm v10.4+
Rocketchat: a bot user with password, a channel that the bot user can join

### Run locally:
```bash
# fill in the env vars:
cp .env.sample .env
# run the app:
npm install
npm run build
npm run start
```

To test the app:
- head to the channel
- message starting with `!Rocky`
